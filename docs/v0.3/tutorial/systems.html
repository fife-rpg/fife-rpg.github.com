SystemsÂ¶

Besides entities and components there is a third part of the component based entity system: Systems. Basically FIFErpg Systems work the same as bGrease Systems . The only addition is that they use the same registering methods as Components and Actions. The manager can be accessed with:

from fife_rpg.systems import SystemManager

The setting name in the settings file is “Systems”
